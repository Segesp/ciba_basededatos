## 🐛 Si Algo No Funciona

### Problema: Servidor no inicia
**Solución:**
```bash
npm install
npm start
```

### Problema: Puerto 5000 ocupado
**Solución:**
```bash
# Cambiar puerto en .env
echo PORT=3000 > .env
npm start
```

### Problema: Base de datos vacía
**Solución:**
- Crea el primer evento desde el frontend
- Se inicializará automáticamente

### Problema: API desconectada
**Solución:**
1. Abre DevTools (F12)
2. Revisa si hay errores en Console
3. Verifica que servidor dice "ejecutándose en http://localhost:5000"
